Configuração de FTP

Autor: Bernardo Carneiro

Iniciar -> Ferramentas Administrativas -> Gerenciador do Serviço de Informações da Internt (ISS) 6.0 ->

SERVER01 (Nome do seu servidor) -> Sites FTP -> Clique com o botão direito -> Novo Site FTP ->

Por exemplo: Documentos do SERVER01 -> Avançar -> Digite o IP 192.168.0.1 (O IP do seu servidor) ->

Porta 21 -> Avançar -> Não isolar usuários -> Avançar -> C:\inetpub\ftproot\server01 -> Avançar ->

Permitir permissões -> Leitura e Gravação -> Avançar -> Concluir

Na mesma aba, documentos do SERVER01 -> Clique com o botão direito -> Propriedades



Iniciar -> Ferramentas Administrativas -> DNS -> Zonas de pesquisa direta -> server01.com -> Clique com o botão direito ->

Novo alias (CNAME) -> Nome ftp -> Procurar -> Igual a pasta pai -> OK -> OK



Teste FTP - Máquina Windows XP

Iniciar -> Prompt de Comando -> Digite o seguinte código: ftp ftp.server01.com

O terminal irá solicitar o usuário e senha. Você estará logado dentro do servidor FTP.
